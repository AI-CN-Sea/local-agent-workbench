# Roadmap

## v0.1.x

- Stabilize the runnable local demo.
- Keep README, screenshots, and examples aligned with the UI.
- Improve smoke coverage for TaskContract, Agent Run, privacy, and localhost Ollama paths.
- Strengthen medium-risk approval with prompt-hash-bound approval tokens.
- Continue refining Chinese/English UI wording without changing the core workflow.

## v0.2.x

- Improve artifact preview without automatic file parsing.
- Add stronger Skill Manifest validation for metadata-only skill packages.
- Improve memory review, disable, and export controls.
- Improve Agent Run pause, resume, retry, cancel, and recovery UX.
- Add richer local model routing metadata and local cost/token summaries.

## v0.3.x

- Add optional sandboxed file parsing behind explicit user approval.
- Add multiple local provider profiles beyond Ollama if they can preserve local-only boundaries.
- Add stricter tool governance and approval policies.
- Add better artifact center workflows.

## Deferred

- External model APIs.
- Web search.
- MCP external retrieval.
- Desktop software automation.
- Heavy PDF/OCR processing.
- Any workflow requiring API keys or cloud credentials by default.
